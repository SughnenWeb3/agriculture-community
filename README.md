# Agriculture Community Website

A simple, responsive agriculture community website with user authentication features.

## Features

- User registration and login
- Password reset functionality
- Mobile-responsive design
- Clean, simple UI with vanilla CSS
- Smooth navigation between pages

## Project Structure

```
agriculture-community/
├── css/           # Stylesheets for each page
├── html/          # HTML pages
├── img/           # Images and assets
└── README.md      # Project documentation
```

## Pages

- **index.html** - Landing page
- **login.html** - User login
- **register.html** - User registration
- **forgotPassword.html** - Password reset request
- **forgotPassword1.html** - Email verification code
- **setNewPassword.html** - Set new password
- **success.html** - Password reset success

## Technologies Used

- HTML5
- Vanilla CSS
- Responsive Design
- Box Icons

## Getting Started

1. Clone this repository
2. Open `html/index.html` in your browser
3. Navigate through the application

## Navigation Flow

1. Start at index.html
2. Go to login.html or register.html
3. Use forgot password flow if needed
4. All pages are linked with back buttons for easy navigation
